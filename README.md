# E-commerce-site-UI
A complete E-Commerce Website UI mockup featuring a modern homepage, product listing, cart, checkout flow, and seller dashboard. Designed to showcase UI/UX skills with clean layouts, intuitive navigation, and polished component structures.
A fully designed E-Commerce Website UI Mockup showcasing essential user flows and modern interface components. This design includes:
Homepage with hero banner, categories, featured products, and promotional sections (as seen on page 1 of the PDF mockup) 
Product Listing Page featuring filters, categories, product cards, ratings, and price elements (page 2 visuals) 
Shopping Cart UI with product previews, pricing, quantity controls, and summary details
Checkout UI including address selection, payment options, delivery details, and final order review
Seller Dashboard with analytics, product management, order table, and insights
This project highlights:
Responsive and clean layout structure
Professional component design
Modern color themes and typography
Practical e-commerce UX patterns
Strong UI/UX understanding suitable for real-world applications
